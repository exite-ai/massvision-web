### 3.2 初期メイン画面（グリッド＋メニューUI）
- ヘッダー

  - ユーザーメニュー（ログイン・設定・ヘルプ）
- メイン
  - 左：グリッドキャンバス（SVG/Canvas、青いグリッド＋対角線、キャラクタ配置可、ズーム・パン対応）
  - 右：メインメニュー（番号キーまたはクリックで選択）
    1. マクロ編集
    2. 配置
    3. リンクスライド
    4. 環境設定
    5. データ管理
    6. version情報
    7. 終了（ログアウト/アプリ終了）
- フッター
  - ショートカットキーやヒント表示
  - バージョン情報
- [デザイン例]
  - 画面左にグリッド、右に縦型メニュー
  - メニューはキーボード操作・クリック両対応
